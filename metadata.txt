about=AgenticGIS adds a chat dock to QGIS. Type a request in natural language and an LLM agent carries it out by generating and running PyQGIS code, giving it access to QGIS, Processing, project layers, and installed plugins. Requirements: QGIS 3.22 or newer and an LLM connection using a provider API key, compatible custom endpoint, or installed local CLI agent. The plugin has no bundled Python package dependencies. Google Earth Engine features are optional and require the Google Earth Engine QGIS plugin to be installed and authenticated separately.
